# Politica de Privacidade

Status: modelo inicial para revisao juridica.

1. Coletamos apenas dados necessarios para autenticacao e uso da plataforma.
2. Dados de pagamento sao processados pelo provedor externo (Mercado Pago).
3. Telemetria anonima e opt-in.
4. O usuario pode solicitar atualizacao ou exclusao de dados conforme legislacao aplicavel.
5. Segredos e credenciais sao protegidos por boas praticas de seguranca.
